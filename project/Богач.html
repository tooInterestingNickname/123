Богач

Описание

Богач является отличным выбором для уборки по дому. Обладает хорошим чувством юмора. С ним уборка покажется развлечением.

Спектр услуг

  • Уборка по дому без использования хим. средств
  • Уборка по дому с использованием хим. средств
  • Ремонт бытовой техники
